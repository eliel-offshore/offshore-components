# Debugging

<div className="subheading">Using VS Code</div>

Download and install Chrome debugger
Create a <code>launch.json</code> to configure VS Code debugger
And put it inside <code>.vscode</code> in root 

<Source 
  language='css'
  code={dedent`
     {
          "version": "0.2.0",
          "configurations": [
              {
                  "name": "Chrome",
                  "type": "chrome",
                  "request": "launch",
                  "url": "http://localhost:3000",
                  "webRoot": "${workspace}/src"
              }
          ]
      }
  `}
/>

Go to debug mode

Run <code>ctrl + shift + d</code>

<img src={image.src} />

Add a breakpoint into your vscode and start debugging.
